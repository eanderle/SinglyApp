GMaps.js — Basic

Using GMaps.js is as easy as:

new GMaps({
  div: '#map',
  lat: -12.043333,
  lng: -77.028333
});

You must define container ID, latitude and longitude of the map's center.

NoteYou also can define zoom. By default, zoom is 15.
